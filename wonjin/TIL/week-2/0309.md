# 자동완성 3총사

> - importantForAutofill, autoComplete, textContentType

- `importantForAutofill`

  > 키체인 자동완성 (android)

- `autoComplete`

  > sms-otp - 인증문자 자동완성

- `textContentType`

  > 키체인 자동완성 (ios)

- `keyboardType`

  > 키보드 자판타입 변경
  > decimal-pad = 숫자패드
  > email-address = @포함된 키보드

- `returnKeyType`

  > 키보드 배열에서 완료표시 변경

- `onSubmitEditing`

  > 엔터나 탭을 누른 후(섭밋 이 후) 동작하는 액션

- `blurOnSubmit`

  > 키보드 내려가는거에 관한 옵션

- `clearButtonMode`
  > 입력중 삭제 기능

```
Property 'trim' does not exist on type 'never'
```
